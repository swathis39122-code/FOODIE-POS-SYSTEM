===========================================
IMAGES FOLDER - README
===========================================

Place your restaurant images in this folder.

RECOMMENDED IMAGES:
-------------------
1. hero-bg.jpg (1920x1080px)
   - Used for the home page hero section
   - Should be a high-quality food or restaurant image

2. Gallery Images (800x800px or square format)
   - Food photos
   - Restaurant interior photos
   - Chef photos
   - Any other relevant images

IMAGE FORMATS:
-------------
- JPG or PNG format
- Optimize images for web (use tools like TinyPNG)
- Keep file sizes reasonable for fast loading

CURRENT STATUS:
--------------
The website currently uses placeholder images from Unsplash.
Replace them with your actual images for the best results.

HOW TO UPDATE:
-------------
1. Add your images to this folder
2. Update the image paths in gallery.html
3. Update the hero background in css/style.css or index.html
